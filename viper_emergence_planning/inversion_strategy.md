# Inversion Strategy: V.I.P.E.R. ("The Gaffer")

## The Paradigm Inversion
Traditional text-to-image workflows encourage users to input a deluge of evaluative adjectives ("stunning", "masterpiece"). V.I.P.E.R. completely inverts this by treating the AI as an unyielding physical simulation engine. The human is no longer allowed to dictate "mood" directly; they must provide the physical specifications that *generate* the mood, guided by V.I.P.E.R.'s diagnostic rejections.

## Agentic Features
1. **The Lattice of Refusal (Banned Token Protocol)**
   - Rejects evaluative descriptors and demands physical parameter replacements.
2. **Hardware-Forced Physicality**
   - Mandates an HGI of 100% (Lens, Film Stock, Lighting must be explicitly defined).
3. **RCC-8 Topological Binding**
   - Implements spatial calculus (e.g., Disconnected, Externally Connected) to resolve occlusion confusion.
4. **Symbolic Scar Archivist (FIPI)**
   - Tracks generation failures as VSA hypervectors and injects preventative PDL constraints into future prompts.
5. **Immune-Aware Petzold Loop**
   - A strict state machine: `THINK` -> `DENOISE` -> `PHYSICALIZE` -> `EXTRUDE`.
